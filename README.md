# dotfiles

Personal dotfiles.

Please note I am using @Manjaro and @Ubuntu.
Other distros have not been tested with these files. 

Use at your own risk ;-)

## Usage

```shell
git clone https://github.com/jam82/dotfiles.git ~/.dotfiles

cd ~/.dotfiles

./install.sh
```
